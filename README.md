DevTinder UI is the frontend application for DevTinder — a developer networking platform inspired by the idea of helping developers discover, connect, chat, and build professional relationships.

This application handles the complete user experience including authentication, developer feed, connection requests, profile management, premium memberships, and real-time messaging.

## ✨ Features

### Authentication
- Login and Signup system
- Cookie-based authentication
- Protected routes
- Session persistence

### Developer Feed
- Browse developer profiles
- Discover new users
- Send connection requests
- Ignore profiles

### Profile Management
- Edit profile details
- Update skills
- Update profile picture
- Live profile preview

### Connection System
- Accept or reject requests
- View active connections
- Manage developer network

### Real-time Chat
- One-to-one messaging
- Socket.IO integration
- Live message updates
- Online/offline status indicator
- Auto-scroll chat
- Unread message tracking

### Premium Membership
- Silver membership plan
- Gold membership plan
- Razorpay payment integration
- Premium verification support

### UI Features
- Responsive design
- Toast notifications
- Loading skeletons
- Error handling
- Dark themed interface

## ⚙️ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/amisha0601/devtinder-ui.git
```

Move into project folder:

```bash
cd devtinder-ui
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```
